devops 工具汇总

发表于 2018-11-12 | 分类于 devops | 阅读次数:

介绍

运维相关工具的汇总记录。

涉及的工具内容：

  1. 中间件的运维，部署

  2. db的运维，监控工具

  3. 进程/内存的运维，监控工具

中间件

redis

监控工具：主要依赖redis-cli，例如：redis-cli -p 7000 info | grep ops

分析大key：redis-cli -p 7000 –bigkeys

dump rdb，分析key分布，内存大小等：https://github.com/sripathikrishnan/redis-rdb-tools

rocketmq

监控工具：rocketmq console

功能：查看broker节点状态，主从信息，in，out tps，创建topic，观察实时connection。

elasticsearch

_plugin/head：概览，索引，基本查询，符合查询

_plugin/kopf：内存大小，磁盘占用，索引管理。

zookeeper

可视化：taokeeper-monitor

DB

mysql

mysql的可视化监控工具：percona

提供功能：cpu，mem，disk r/w， network io， db conns， qps， ram， uptimes

进程/内存

jemalloc：用于对外内存leak分析，监听linux native方法调用情况，非侵入；

jvisualvm：实时查看堆栈信息，需要添加jmx启动参数

jprofiler：dump prof文件，分析类的instance， size，并拿到其调用链。可在运行时jmap dump，亦可在oom时自动dump，增加启动参数即可。
