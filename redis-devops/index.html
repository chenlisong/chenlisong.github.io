redis的常用命令：info，monitor，scan

版本：>= 4.0

info

包含几块内容

  • Server redis服务相关信息，版本号，系统版本，id，端口等

  • Clients 客户端连接信息，连接数等

  • Memory 内存相关信息，已使用的内存，总内存等

  • Persistence 持久化，rdb，aof相关配置

  • Stats 状态，

  • Replication 主从复制信息

  • Cpu cpu计算量统计信息

  • Cluster 集群信息

  • Keyspace key统计信息
